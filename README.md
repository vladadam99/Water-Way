# Water Way

**Water Way** is a Shopify ecommerce brand for water-saving, water-filtration, cleaning, irrigation, pumps, rainwater harvesting, leak detection, and related water products.

## Current build status

This repository contains:

- a lightweight Shopify Online Store 2.0 starter theme shell;
- Water Way brand guidelines;
- page copy for About, FAQ, Shipping, Returns, Privacy, and Terms;
- a draft product import CSV for the first Water Way catalogue batch;
- collection planning and smart-collection tag rules;
- operating instructions for connecting Shopify, GitHub, and future automation.

## Shopify connection target

Store: `epj3de-4y.myshopify.com`  
Market: United Kingdom  
Currency: GBP  
Brand: Water Way

## Important launch rule

Do not publish products for sale until supplier source, cost, stock, shipping terms, and images are confirmed.

## Repository structure

Shopify theme files live at repository root because Shopify's GitHub theme integration expects the default theme folder structure:

- `assets/`
- `config/`
- `layout/`
- `locales/`
- `sections/`
- `snippets/`
- `templates/`

Business planning and import files live in:

- `docs/`
- `catalog/`
- `scripts/`

## First action checklist

1. Upload this repository content to `vladadam99/Water-Way` on the `main` branch.
2. In Shopify Admin, connect the `main` branch as an unpublished theme.
3. Import `catalog/water-way-products-v1-draft.csv` as a draft product batch.
4. Review products, suppliers, images, margins, shipping weights, and legal policies.
5. Publish the theme only after preview testing.
