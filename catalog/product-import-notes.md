# Product Import Notes

File: `water-way-products-v1-draft.csv`

This is a draft catalogue seed, not final live inventory.

## Why products are draft
The first batch is intentionally set to:

- `Status = draft`
- `Published on online store = false`
- `Inventory quantity = 0`

This prevents accidental sales before suppliers, stock, fulfilment, product images, exact specifications and margins are confirmed.

## Import steps
1. Shopify Admin > Products.
2. Click Import.
3. Upload `catalog/water-way-products-v1-draft.csv`.
4. Review import preview.
5. Import products.
6. Add images and supplier details.
7. Review product weights, compatibility, prices and policies.
8. Activate products only after review.
