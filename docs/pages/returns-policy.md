# Returns and Refunds Policy

This returns policy is a starter template and should be reviewed before launch.

## Returns
Customers may request a return for eligible unused products within the applicable returns period. Products should be returned complete, unused and in their original packaging.

## Non-returnable items
Some products may not be returnable once used, installed, cut to size, opened for hygiene reasons or specially ordered. Product pages should state any item-specific return restrictions.

## Faulty or damaged products
If a product is faulty or arrives damaged, customers should contact Water Way with their order number and clear photos. We will review the issue and advise the next steps.

## Compatibility
Customers should check fitting sizes, measurements and compatibility before ordering. We recommend contacting Water Way before purchase if unsure.

## Refunds
Approved refunds are normally returned to the original payment method. Refund timing depends on the payment provider.

## Return shipping
Return shipping responsibility depends on the reason for return and applicable consumer law. This section should be finalised before launch.
