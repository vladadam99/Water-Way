# Shopify + GitHub Setup

## Goal
Connect `vladadam99/Water-Way` to Shopify as an unpublished theme first. Preview and test before publishing.

## Steps
1. Upload this repository content to GitHub on the `main` branch.
2. In Shopify Admin, go to **Online Store > Themes**.
3. In **Theme library**, choose **Add theme > Connect from GitHub**.
4. Select the GitHub account `vladadam99` and repository `Water-Way`.
5. Select branch `main`.
6. Add it as an unpublished theme.
7. Preview it.
8. In the theme editor, adjust homepage sections and save.
9. Confirm that Shopify commits the saved change back to GitHub.
10. Publish only after preview testing.

## Important
Shopify's GitHub integration syncs theme code both ways. Changes made in Shopify admin can be committed back to GitHub automatically.
